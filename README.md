# Digital Certificate Automation (PowerShell)

## 📌 Overview
This project automates the installation and management of digital certificates (.PFX) on Windows systems using PowerShell.

## 🚀 Features
- Automated certificate installation
- Secure password handling
- Error handling and validation
- Displays certificate details (Thumbprint and Expiration Date)

## 🛠️ Technologies
- PowerShell
- Windows Certificate Store

## ▶️ Usage

```powershell
.\install-certificate.ps1 -caminhoPfx "C:\certificado.pfx" -senha "password"
